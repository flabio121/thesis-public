# Public Reproducibility Notes

This public repository is a minimal thesis-document archive. The COMSOL-first modeling workflow, raw exports, and private lab data are intentionally excluded.

The included PDF is the canonical public viewing copy. The included LaTeX source can be rebuilt if the required LaTeX packages are available and the referenced public figures/tables remain in place.

Quantitative claims in the thesis should be traced to the private working repository and the COMSOL model/export history. Generated conceptual schematics are supporting communication figures and should not be interpreted as measured data.
